# Capsule Toy Collection PWA

A mobile-first PWA for managing real capsule toys and gacha-gacha collections.

## Features

- Item create/list/detail/edit/delete
- Series management, completion rate, duplicate tracking
- Photo storage in IndexedDB as Blob data
- Automatic image compression
- Search, filters, and sorting
- Statistics
- JSON backup / import including images
- Offline support
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
